# 🔥 **Dark-FS**

**Dark-FS is a cross-platform CLI file manager for Windows, Linux, and Termux.**

---

## 🚀 **Features**
- ⚡ **CLI file navigation tool**
- ⚡ **Fast terminal operations**
- 🎨 **Colored output using pyfiglet and colorama**
- 🍃 **Lightweight and simple design**

---

## 📦 **Installation**
```bash
pip install dark-fs
```

---

## 💻 Usage
```bash
dfs
```
